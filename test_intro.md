# Самостоятельная работа.

**Вы сдаете мне файл YourName_intro.R с коммантированным кодом, в котором последовательно выполняете следующие задания:**

1. Создайте переменную name с вашим именем
2. Создайте вектор odd, в котором три раза повторяется последовательность нечетных цифр (1-9).
3. Создайте вектор honest, состоящий из четных цифр (0-8), каждая из которых повторяется 4 раза.
4. Какое число элементов в векторах odd и honest?
5. Вам необходима функция haploNet из пакета pegas. Что необходимо сделать, чтобы ею воспользоваться? Вызовите справку по данной функции.
6. Загрузите данные с систематикой видов рыб Средиземного моря:  Fish taxonomy for national monitoring.csv: https://github.com/sophianazarova/R-for-bioinformatics/blob/main/Fish%20taxonomy%20for%20national%20monitoring.csv
7. Загрузите данные по мониторингу рыб Средиземного моря: NM and TAU 2008-2023.xlsx - лист "Length data": https://github.com/sophianazarova/R-for-bioinformatics/blob/main/NM%20and%20TAU%202008-2023.xlsx
8. Какого типа объект вы создали из таблицы "Length data"?
9. Посмотрите, какого типа данные представлены в столбцах таблицы "Length data"?
10. В каком из столбцов таблицы "Length data" есть пропущенные значения?
11. Какого вида рыба самая тяжелая в 2015 году в осеннем улове на глубине 80 метров? Сколько она весила и какая у нее длина тела?
12. Оцените по доступным данным суммарную биомассу рыб в каждом трале (NetID).
13. Добавьте в таблицу "Length data" информацию о том к какому классу (Class) и семейству (Family) относится каждый из видов рыб, их коммерческое значение (Commercial) и происхождение (Origin).
14. Сделайте таблицу с количеством видов каждого семейства каждый год весной и осенью в формате: Год, сезон, семейство, количество видов
15. Трансформируйте таблицу, полученную в задании 13 так, чтобы каждое семейство было в отдельном столбце таблицы.
16. Сохраните таблицу, полученную в задании 14 в файл Excel.
17. Посчитайте количество и биомассу каждого вида рыб в каждом трале (NetID)
18. Посчитайте средний суммарный годовой улов коммерческих инвазивных видов рыб
19. Нарисуйте динамику количества видов рыб из класса хрящевые рыбы (Elasmobranchii). Сохраните рисунок в формате PDF.
20. Нарисуйте размерно-частотное распределение вида Boops boops весной 2020 года. Сохраните рисунок в формате JPG.
